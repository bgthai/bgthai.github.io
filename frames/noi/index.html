คำปรารภ
บทที่ หนึ่ง :  สำรวจกองทัพที่สมรภูมิคุรุคเชทระ
บทที่ สอง :  บทสรุป ภควัต-คีตา
บทที่ สาม :  คารมะ-โยกะ
บทที่ สี่ :  ความรู้ทิพย์
บทที่ ห้า :  คารมะ-โยกะการปฏิบัติในคริชณะจิตสำนึก
บทที่ หก :  ดยานะ-โยกะ
บทที่ เจ็ด :  ความรู้แห่งสัจธรรม
บทที่ แปด :  การบรรลุถึงองค์ภควาน
บทที่ เก้า :  ความรู้ที่ลับสุดยอด
บทที่ สิบ :  ความมั่งคั่งแห่งสัจธรรม
บทที่ สิบเอ็ด :  รูปลักษณ์จักรวาล
